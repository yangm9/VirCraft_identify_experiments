# Training Data Sources for Viral Identification Tools

Grab training data for all machine/deep learning tools and dereplicate.

- **VirSorter2**: high-quality non-RefSeq genomes (excluding ssDNA viruses, Varsani unpublished data); available at Zenodo (https://zenodo.org/record/4297575).
- **Vibrant**: training sequence identifiers listed in Supplementary File 8 (10.1186/s40168-020-00867-0).
- **VirFinder & DeepVirFinder**: share the same training set derived from NCBI RefSeq; described in Additional file 2: Table S2 (10.1186/s40168-017-0283-5).
- **geNomad**: training and evaluation reference sequences available at https://doi.org/10.5281/zenodo.8049246.
